QUESTIONS

Straight answers

Is Freelancer Xito Events free?

+

Yes, completely. The app is free to download for freelancers and event production staff.

How do I download the app?

+

You can download the APK package directly using the "Download APK" button above.

Do I need an account to use the app?

+

Yes, an authorized account is required to log in and access event assignments, invoices, and folders. You can sign up in the app or get invited by an agency.

Does it support Nepali event dates?

+

Yes! The app features a native Nepali calendar (Lagan) with an integrated date converter to help you manage wedding seasons.

Can I manage event files and albums?

+

Yes. The built-in Xito Drive and Xito Album modules let you upload event photos, select favorites, and track video/photo editing progress in realtime.

How do I track event finances?

+

The app features a Finance module where you can record client advances, partial payments, payouts, and track balances on every contract.
